직장인 관상 샘플러 - 오디오 자동재생 오류 수정 버전

수정 내용:
- 친구들이 GitHub Pages 링크로 접속했을 때 뜨던
  "브라우저가 오디오 재생을 막았습니다" 팝업을 줄이도록 수정했습니다.
- 관상 분석하기 버튼을 누르는 순간, 랜덤으로 선택된 mp3를 미리 깨워둔 뒤
  3초 후 같은 오디오를 재생합니다.
- 팝업 alert 대신 화면 안내 문구로 바꿨습니다.

동작 방식:
1. 카메라 켜기 버튼을 누릅니다.
2. 관상 분석하기 버튼을 누릅니다.
3. 카메라 화면 위에 "분석중..."이 3초간 표시됩니다.
4. 3초 후 5개 멘트 중 하나가 랜덤으로 재생됩니다.
5. 멘트 자막은 카메라 화면 아래쪽에 직접 표시됩니다.
6. 아래의 Gen Z 스타일 이미지 카드도 결과에 맞게 바뀝니다.

GitHub Pages 업로드:
- index.html
- assets 폴더 전체
를 저장소 루트에 업로드하세요.

주의:
- iPhone Safari, 일부 인앱 브라우저, 카카오톡/인스타그램 내장 브라우저는 소리 재생 정책이 더 엄격할 수 있습니다.
- 친구에게 링크를 보낼 때는 카카오톡 내부 브라우저보다 Chrome/Safari에서 열라고 안내하면 더 안정적입니다.
