직장인 관상 샘플러 - 친구 휴대폰 소리 재생 수정 버전

변경한 부분:
- 디자인, 카메라 화면, 분석중 3초, 랜덤 결과, 자막, Gen Z 카드 흐름은 그대로 두었습니다.
- 오디오 재생 방식만 바꿨습니다.
- 관상 분석하기 버튼을 누르는 순간 선택된 mp3를 무음으로 먼저 재생해두고,
  3초 뒤 결과가 나올 때 같은 오디오를 처음으로 되감은 뒤 소리를 켭니다.
- iPhone Safari/Chrome에서 3초 뒤 audio.play()가 자동재생으로 막히는 문제를 줄이기 위한 방식입니다.

GitHub 업데이트:
1. 이 ZIP 압축을 풉니다.
2. GitHub 저장소 루트에 아래 파일을 덮어쓰기 업로드합니다.
   - index.html
   - README.txt
   - assets 폴더 전체
3. Commit changes를 누릅니다.
4. Pages 반영 후 링크 뒤에 ?v=3 을 붙여서 새 버전으로 테스트하세요.

예시:
https://내아이디.github.io/office-face-sampler/?v=3

- iPhone 무음 모드가 켜져 있으면 소리가 안 날 수 있습니다.
- 카카오톡/인스타그램 내장 브라우저보다 Safari 또는 Chrome에서 여는 것이 안정적입니다.
